Introducción a HTML

HTML es el lenguaje de marcado estándar para crear páginas web. Con HTML, puedes definir la estructura y el contenido de una página web utilizando una variedad de elementos y etiquetas.

Elementos Básicos de HTML

Algunos elementos básicos de HTML incluyen:

  • <html>: Define el documento HTML.
  • <head>: Contiene metadatos del documento.
  • <title>: Define el título de la página.
  • <body>: Contiene el contenido visible de la página.

Enlaces en HTML

Los enlaces en HTML se crean utilizando la etiqueta <a>. Por ejemplo:

<a href="https://www.example.com">Enlace de ejemplo</a>
